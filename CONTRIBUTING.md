# Contribuir

1. **Node 20+** y herramientas de compilación nativas si estás en Windows (ver README raíz).
2. Cloná el repo y ejecutá **desde la raíz** `npm install` (instala `bazar-app` vía `postinstall`) o, si preferís, `cd bazar-app && npm install`. Luego `npm run dev` desde la raíz o desde `bazar-app/`.
3. Antes de un PR: `npm run lint` y, si tocás build, `npm run build:vite`.
4. Mantené los cambios acotados al problema o feature; evitá commitear artefactos de `release/` o credenciales.
