# shadcn/ui

Instalación (desde esta carpeta `bazar-app`):

```bash
npx shadcn@latest init
npx shadcn@latest add table dialog drawer command badge input button select tabs card toast popover tooltip
```

Los componentes se copiarán aquí (`src/components/ui/`). Tailwind v4 ya está integrado vía `@tailwindcss/vite` y `src/index.css`.
